CEBS (College Event Booking System) is a web-based platform designed to simplify and digitize college event management. The system allows students to explore and register for events, make secure online payments, receive automated notifications, and use QR-based entry verification for seamless participation.

The platform includes dedicated dashboards for students, faculty coordinators, and administrators, along with analytics, reports, authentication systems, and role-based access control for efficient event management.

🔹 Features:
• Online Event Registration

• Razorpay Payment Integration

• QR Code Based Event Entry

• Email, SMS & WhatsApp Notifications

• Analytics & Reports Dashboard

• Role-Based Access Control

• OTP Authentication & Password Reset

• Responsive UI for Desktop and Mobile Devices

🔹 Tech Stack:
HTML5, CSS3, JavaScript, Bootstrap 5, PHP, MySQL, AJAX, jQuery, Razorpay, Cloudinary, PHPMailer, Dompdf, PhpSpreadsheet, UltraMsg, Google Maps API.
